List of HTML Tags

  1. <html>:It is the root element of an HTML document.
  2. <head>:It Contains meta-information about the document.
  3. <title>:It Sets the document's title.
  4. <meta>:It Provides metadata about the document.
  5. <link>:It is Used to link external resources.
  6. <script>:It Embeds or references JavaScript code.
  7. <style>:It Contains CSS code or links to stylesheets.
  8. <body>:It Encloses the main content.
  9. <h1> to <h6>:It is used for Headings of varying levels.
  10. <p>:It Represents a paragraph of text.
  11. <a>:It Creates hyperlinks.
  12. <img>:It Embeds images.
  13. <ul>:It Defines an unordered list.
  14. <ol>:It Defines an ordered list.
  15. <li>:It Represents list items.
  16. <div>:It is a generic container element.
  17. <span>:It is a Inline container for styling or scripting.
  18. <form>:It Defines a container for user input elements.
  19. <input>:It Represents form controls.
  20. <textarea>:It Creates a multi-line text input field.
  21. <button>:It Defines a clickable button.
  22. <table>:It is Used to create tables.
  23. <tr>:It is the Table row element.
  24. <td>:It is the Table cell element.
  25. <th>:It is the Table header element.
  26. <iframe>:It Embeds external web content.
